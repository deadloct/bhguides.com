# AOMINE's Patcharrior Speed Run

AOMINE's 14 minute speed run included the following novel approaches:

* Patcharrior for the 4sp, which provides a massive fury and recoils.
* Sardinex Alternator, which boosts familiar damage.
* Supercharged Poly to 2x that extra boost.
* Possible new uses of some augments.

## Familiars 

Patcharrior:

* Pumps: 3x Barrier
* Brain: Attack team when you get hit
* Bone: Increase dmg of teammates behind you
* Chip: Overheal converts to shields (chip didn't matter much)

Lerky

* Pumps: 2 Empower, 1 Speed
* Brain: Attack weakest when you hit an enemy
* Bone: Increase dmg while above 75% HP
* Chip: Increase dmg of SP skills
