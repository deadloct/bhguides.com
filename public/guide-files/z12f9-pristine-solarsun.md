# Pristine's Solarsun Guide

Solarsun, Solarsun is the "mini boss" from the 9th Flag of Zord Attacks zone.

Kinda tricky to fight him because he has "Damage Furthest, 2 times" skill, BUT it cost him 2sp. So we have chance to kill him before let him using the 2sp skills repeatly. 

I'm playing as DPS at 7.7k Total Stats, using 3/4 Camouflage set T11, 2/4 Lucernas set T12, and 1/3 Mystery set T11, and Karlorr as Offensive Pet that's my equipment gear. 

As you guys can see, my gear is not proper enough (in terms of Tier, Upgrade or even complete set). So maybe it will be easier if u have fully proper set.
